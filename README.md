# utiler
A Tiling Utility

Useful for generating Textures/Heightmaps when importing UE4 Landscape Tiles

## Demo 


<img align="center" width="480" height="480" src="./res/color.png">

```shell
  python3 ../utiler.py ../res/color.png -s 1 -d 2017 -f "../res/Tile_{x}_{y}.png" 
```
